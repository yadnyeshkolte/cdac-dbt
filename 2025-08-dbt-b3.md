## MySQL Table Creation and Data

### Table: Books

```sql
CREATE TABLE Books (
    book_id INT PRIMARY KEY,
    title VARCHAR(100),
    author VARCHAR(50),
    price DECIMAL(10,2),
    published_year INT
);
```

### Table: Borrowers

```sql
CREATE TABLE Borrowers (
    borrower_id INT PRIMARY KEY,
    name VARCHAR(50),
    book_id INT,
    borrow_date DATE,
    return_date DATE,
    FOREIGN KEY (book_id) REFERENCES Books(book_id)
);
```
